# WordCrab
WordCrab is a smart vocabulary learning app that helps users learn English words through quizzes, auto-translation, speech pronunciation, and spaced repetition (SRS) for long-term memory.
# 🦀 WordCrab

WordCrab istifadəçilərə ingilis sözlərini öyrənmək üçün hazırlanmış interaktiv veb tətbiqdir. Tətbiq avtomatik tərcümə, quizlər və SRS (spaced repetition) sistemi vasitəsilə öyrənməni daha effektiv edir.

## 🚀 Xüsusiyyətlər
- 🌍 Avtomatik tərcümə
- 🧠 SRS (təkrarlama sistemi)
- 📝 Quiz sistemi
- 🔊 Tələffüz (səsləndirmə)
- 📊 Statistikalar və inkişaf izləmə
- 💾 Offline məlumat paylaşımı
